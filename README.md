# Darsu Food Cafe

**Status:** Work in Progress

A modern and responsive cafe web application built using HTML, CSS, and JavaScript. This project is currently under development, with new features and pages being added regularly.

---

## Overview

**Darsu Food Cafe** is a front-end cafe website based in Belbari-3, Morang, Nepal. The application allows users to log in with different roles and browse the cafe's offerings through a clean and user-friendly interface.

---

## Current Features

### Homepage

* Personalized greeting header
* Cafe branding and location display
* Food and table search bar

### Login Page

* Role-based login system (Admin, Member, Visitor)
* Email and password login form
* Google Sign-In button
* Links to Register and Forgot Password pages

### Additional Features

* Responsive design across all pages
* JavaScript-based login interactivity
* Role-switching functionality

---

## Technologies Used
